# DevBookmarks 📂

## ✨ Introducción

Navegando por internet, siempre me encuentro con alguna que otra página web útil, una librería interesante, catálogos de componentes, iconos, SVGs, y un sinfín de recursos para el desarrollo. Pero aquí está el problema:

- Si los guardo en "Favoritos" del navegador, se me hace un desastre.
- A veces los pierdo entre tantas cosas guardadas.
- Otras veces simplemente me digo "luego lo miraré", pero cuando llega ese "luego", no recuerdo ni dónde lo vi.
- Y si me acuerdo, no tengo ni idea de cómo encontrarlo rápidamente.

Después de muchas veces pasar por esto, decidí crear **este repositorio en GitHub**. Es un espacio donde puedo almacenar, categorizar y organizar todos estos enlaces útiles en un solo lugar. GitHub es una herramienta versátil, con control de versiones y fácil acceso, lo que lo convierte en la mejor opción para mantener estos recursos bien organizados.

## 🔄 Organización del repositorio

Cada sección de este repositorio está dividida en archivos Markdown independientes dentro de carpetas organizadas. Esto me permite acceder rápidamente a lo que necesito sin tener que escarbar entre miles de enlaces desordenados.

**Importante:** La idea de este repositorio **no es** simplemente almacenar enlaces a documentaciones oficiales de tecnologías populares como Angular, React, Vue, etc. Esos recursos son fáciles de encontrar con una búsqueda rápida. En su lugar, aquí guardo **librerías y herramientas menos conocidas**, aquellas que solo encuentras tras investigar bastante y que pueden ser realmente útiles pero no tienen tanta visibilidad.

## 📝 Explicación de las secciones

### 1️⃣ **Por Plataforma o Lenguaje**

Aquí se organizan los recursos según la tecnología o plataforma:

- **Web/** → Tecnologías para el desarrollo web como React, TailwindCSS, Angular, etc.
- **Mobile/** → Tecnologías para desarrollo móvil como React Native, Flutter, Swift, Kotlin.
- **Backend/** → Tecnologías y frameworks de backend como Node.js, Python, Go, Java, etc.

### 2️⃣ **Por Tipo de Recurso**

Estos archivos contienen enlaces y referencias según su categoría:

- **UIComponentLibraries.md** → Catálogos de librerías UI como ShadCN, MUI, Ant Design, Chakra UI.
- **Iconos.md** → Sitios de iconos como Lucide, HeroIcons, FontAwesome.
- **Emojis.md** → Recursos de emojis como Twemoji, OpenMoji.

## 🛠️ Cómo usar este repositorio

1. **Explora las carpetas y archivos Markdown** según lo que necesites. Cada archivo contiene una lista de enlaces con una breve descripción de cada recurso.
2. **Contribuir**: Si encuentras un recurso útil que falta, puedes hacer un _Pull Request_ agregándolo en la sección correspondiente.
3. **Mantenerlo actualizado**: Intentaré actualizar y revisar este repositorio periódicamente para asegurarme de que los enlaces sigan funcionando.

## 📚 Contribuciones

Si quieres ayudar a mejorar este repositorio con más enlaces o categorías, sigue estos pasos:

1. **Haz un fork** de este repositorio.
2. **Agrega o modifica un archivo Markdown** en la carpeta correspondiente.
3. **Haz un Pull Request** con tus cambios.
4. **¡Revisaré y aceptaré las contribuciones útiles!** 🙌

## ✨ Conclusión

Este repositorio me ayudará a tener todos mis recursos en un solo lugar, de una manera organizada y accesible. Si te pasa lo mismo que a mí y sueles perder enlaces útiles, quizás también te sirva.

Si tienes algún comentario o sugerencia, estaré encantado de recibirlo.
